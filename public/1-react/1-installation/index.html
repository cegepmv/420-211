Instatroduction à React

React est une bibliothèque JavaScript populaire développée par Facebook, utilisée pour construire des interfaces utilisateur dynamiques et interactives. Elle facilite la création de composants réutilisables, permettant de structurer et de gérer efficacement les interfaces complexes. Grâce à son modèle de rendu basé sur le DOM virtuel, React garantit des performances optimales et une expérience utilisateur fluide.

Prérequis

Avant d’installer React, assurez-vous d’avoir les outils suivants installés sur votre machine :

  • Node.js : React utilise Node.js pour gérer ses dépendances via npm (Node Package Manager).
  • Un éditeur de code : Comme Visual Studio Code.

Pour vérifier si Node.js est installé, exécutez la commande suivante dans votre terminal :

node -v

Si ce n’est pas le cas, téléchargez et installez Node.js depuis nodejs.org.

Étape 1 : Créer un nouveau projet React

React offre un outil appelé Create React App pour configurer un projet avec une structure de base. Voici les étapes pour démarrer :

1. Ouvrir le terminal

Assurez-vous d’ouvrir un terminal ou une invite de commande sur votre machine.

2. Exécuter la commande d’installation

Dans le répertoire où vous souhaitez créer votre projet, exécutez la commande suivante :

npx create-react-app nom-de-votre-projet
  • Remplacez nom-de-votre-projet par le nom souhaité pour votre projet.
  • L’outil téléchargera toutes les dépendances nécessaires et créera la structure du projet.

Note : npx est un outil inclus avec npm qui permet d’exécuter des packages Node.js sans les installer globalement.

3. Accéder au dossier du projet

Une fois l’installation terminée, naviguez dans le répertoire du projet :

cd nom-de-votre-projet

4. Lancer le serveur de développement

Pour démarrer votre application React, exécutez :

npm start

Cela ouvrira automatiquement votre navigateur et affichera votre application React à l’adresse suivante :

http://localhost:3000

Structure du Projet

Après l’installation, voici les principaux fichiers et dossiers que vous trouverez :

  • src/ : Contient les fichiers source de votre application (composants React).
  • public/ : Contient les fichiers statiques accessibles directement (comme index.html).
  • package.json : Contient les informations sur le projet et ses dépendances.

Étape 2 : Ajouter des dépendances supplémentaires (optionnel)

Vous pouvez installer des bibliothèques supplémentaires en fonction de vos besoins. Par exemple :

  • React Router pour la navigation :
npm install react-router-dom
  • Axios pour les requêtes HTTP :
npm install axios

Étape 3 : Modifier et personnaliser

Vous pouvez maintenant éditer les fichiers dans le dossier src/ pour commencer à construire votre application. Le point d’entrée principal est généralement src/App.js.

Conclusion

React est un outil puissant pour construire des interfaces utilisateur modernes. En suivant ces étapes, vous êtes prêt à démarrer avec React et à développer vos premières applications interactives !
