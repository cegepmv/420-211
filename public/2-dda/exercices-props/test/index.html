JSX et les props - exercices

Voici une série d’exercices basés sur les concepts abordés dans cadre du chapitre 2 ; DDA. Chaque exercice vise à renforcer la compréhension de la transmission des données entre composants et la réutilisabilité des composants.

La section suivante présente une série d’exercices sur l’utilisation des props pour transmettre des données aux composants sous toutes ses formes;

  • les types de bases chaine de caractère int boolean
  • les tableaux et objets en javascript
  • les fonctions

📝 Exercices sur les Props en React

Exercice 1 : Affichage dynamique d’un message de bienvenue

Objectif : Créez un composant WelcomeMessage qui affiche un message de bienvenue avec un nom passé en prop.

Instructions

  1. Créez un composant WelcomeMessage qui prend une prop name et affiche :
    "Bienvenue, [nom] !".
  2. Dans le composant App, utilisez WelcomeMessage avec trois noms différents.

Exemple d’affichage attendu

Bienvenue, Alice !
Bienvenue, Bob !
Bienvenue, Charlie !

Exercice 2 : Afficher l’âge d’un utilisateur

Objectif : Utiliser plusieurs props pour afficher des informations sur un utilisateur.

Instructions

  1. Créez un composant UserInfo qui prend deux props : name et age.
  2. Il doit afficher : "[Nom] a [âge] ans.".
  3. Dans App, affichez UserInfo avec trois utilisateurs différents.

Exemple d’affichage attendu

Alice a 25 ans.
Bob a 30 ans.
Charlie a 35 ans.

Exercice 3 : Vérifier si un produit est abordable

Objectif : Utiliser les props et les expressions conditionnelles dans JSX.

Instructions

  1. Créez un composant PriceTag qui prend une prop price.
  2. Affichez "Cher" si price est supérieur à 100, sinon affichez "Abordable".
  3. Dans App, affichez PriceTag avec plusieurs prix différents.

Exemple d’affichage attendu

Prix : Cher
Prix : Abordable
Prix : Abordable

Exercice 4 : Liste d’achats dynamique

Objectif : Passer un tableau en prop et l’afficher sous forme de liste.

Instructions

  1. Créez un composant ShoppingList qui prend une prop items (un tableau de chaînes de caractères).
  2. Affichez chaque élément sous forme de <li>.
  3. Dans App, passez deux listes différentes à ShoppingList.

Exemple d’affichage attendu

Liste 1 :
- Pommes
- Bananes
- Oranges

Liste 2 :
- Lait
- Pain
- Beurre

Exercice 5 : Affichage d’un profil utilisateur avec un objet

Objectif : Passer un objet en prop et l’afficher dynamiquement.

Instructions

  1. Créez un composant UserProfile qui prend une prop user (objet contenant name, age et city).
  2. Affichez les informations de l’utilisateur dans un <p>.
  3. Dans App, affichez UserProfile avec deux objets utilisateurs différents.

Exemple d’affichage attendu

Alice a 25 ans et habite à Paris.
Bob a 30 ans et habite à Lyon.

Exercice 6 : Transmettre une fonction en prop

Objectif : Passer une fonction en prop et l’exécuter dans un composant enfant.

Instructions

  1. Créez un composant Button qui prend une prop onClick et affiche un bouton <button>.
  2. Lorsque le bouton est cliqué, il doit exécuter la fonction passée en prop.
  3. Dans App, définissez une fonction handleClick qui affiche une alerte "Bouton cliqué !".
  4. Passez handleClick en prop à Button.

Exercice 7 : Affichage dynamique d’une liste d’utilisateurs

Objectif : Générer des composants dynamiquement à partir d’un tableau d’objets.

Instructions

  1. Définissez un tableau users contenant trois objets { id, name, age }.
  2. Créez un composant UserList qui génère dynamiquement des UserInfo avec map().
  3. Affichez UserList dans App.

Exemple d’affichage attendu

- Alice - 25 ans
- Bob - 30 ans
- Charlie - 35 ans

Exercice 8 : Refonte d’un composant réutilisable

Objectif : Transformer une liste statique en composants réutilisables.

Instructions

  1. Créez un composant ListItem qui prend title, description et image en props.
  2. Définissez un tableau reasons contenant des objets { title, description, image }.
  3. Affichez dynamiquement des ListItem à partir de ce tableau.

Exemple d’affichage attendu

[Image] J'aime React : Parce que c'est réutilisable !
[Image] J'aime JavaScript : Parce qu'il est flexible !
[Image] J'aime CSS : Parce qu'il permet de beaux designs !

🚀 Challenge Bonus : Afficher un message dynamique selon l’heure

Objectif : Modifier dynamiquement l’affichage selon l’heure.

Instructions

  1. Créez un composant Greeting qui détermine le moment de la journée (morning, afternoon, evening, night).
  2. Affichez "Good [timeOfDay]!".
  3. Dans App, affichez Greeting et testez-le à différentes heures.

Exemple d’affichage attendu

À 10h : Good morning!
À 15h : Good afternoon!
À 20h : Good evening!
À 23h : Good night!

Exemple d’affichage attendu :

Alice a 25 ans et habite à Paris. Bob a 30 ans et habite à Lyon.

Exercice 2 : la carte utilisateur avec mise en forme conditionnelle

Objectif : Créer une carte utilisateur qui affiche des informations et change dynamiquement de style en fonction de l’âge de l’utilisateur.

Instructions :

  1. Créez un composant UserCard qui prend en props un objet user contenant { name, age, city }.
  2. Si l’utilisateur a moins de 30 ans, affichez un fond vert, sinon affichez un fond bleu.
  3. Dans App, affichez plusieurs UserCard avec des utilisateurs différents.

🔢 Exercice 2 : Liste de produits avec gestion de stock

Objectif : Afficher une liste de produits en précisant s’ils sont en stock ou non.

Instructions :

  1. Créez un composant ProductList qui prend une prop products (tableau d’objets { id, name, price, inStock }).
  2. Pour chaque produit, affichez son nom et son prix.
  3. Si le produit est en stock, affichez "En stock", sinon affichez "Rupture de stock" en rouge.
  4. Dans App, affichez au moins 5 produits.

🔢 Exercice 3 : Filtrage de produits avec une fonction en prop

Objectif : Ajouter une fonctionnalité pour filtrer les produits en fonction de leur disponibilité.

Instructions :

  1. Créez un composant ProductFilter qui prend en prop une fonction onFilterChange.
  2. Le composant doit contenir un bouton "Afficher seulement les produits en stock".
  3. Lorsqu’on clique sur ce bouton, il doit appeler onFilterChange(true), sinon onFilterChange(false).
  4. Dans App, affichez ProductList et mettez en place la logique pour filtrer les produits.

🔢 Exercice 4 : Génération dynamique de composants avec props

Objectif : Générer dynamiquement des composants enfants avec des props basées sur un tableau d’objets.

Instructions :

  1. Définissez un tableau teams contenant plusieurs objets { id, name, members }, où members est un tableau de noms.
  2. Créez un composant TeamCard qui prend name et members en props.
  3. TeamCard doit afficher le nom de l’équipe et une liste des membres.
  4. Dans App, affichez dynamiquement plusieurs TeamCard à partir du tableau.

🔢 Exercice 5 : Passer un composant en prop (Children as Props)

Objectif : Passer un autre composant en prop et l’afficher dynamiquement.

Instructions :

  1. Créez un composant Card qui affiche une boîte avec un titre et n’importe quel contenu passé en prop via children.
  2. Dans App, affichez plusieurs Card contenant des éléments différents (texte, images, boutons).

🔢 Exercice 6 : Génération dynamique de formulaires avec des props

Objectif : Créer un composant FormInput qui génère dynamiquement des champs de formulaire en fonction des props.

Instructions :

  1. Créez un composant FormInput qui prend une prop fields (tableau d’objets { id, label, type }).
  2. Pour chaque élément du tableau, affichez un <label> et un <input> du type correspondant (text, password, email, etc.).
  3. Dans App, affichez FormInput avec plusieurs champs dynamiques.

🔢 Exercice 7 : Passer une fonction de mise à jour en prop (Lifting State Up)

Objectif : Permettre à un composant enfant de modifier l’état du composant parent.

Instructions :

  1. Créez un composant Counter qui prend en prop une fonction onIncrement.
  2. Le composant affiche un bouton "Ajouter 1" et lorsqu’on clique dessus, il appelle onIncrement.
  3. Dans App, affichez un Counter et mettez à jour un état count en fonction des clics.

🔢 Exercice 8 : Liste d’articles avec suppression dynamique

Objectif : Permettre de supprimer dynamiquement des éléments d’une liste.

Instructions :

  1. Créez un composant ArticleList qui prend une prop articles (tableau d’objets { id, title }).
  2. Chaque article affiché contient un bouton "Supprimer".
  3. Lorsqu’on clique sur "Supprimer", l’article est retiré de la liste.

🔫 Challenge Bonus : Dark Mode avec Prop & State

Objectif : Implémenter un mode sombre en utilisant une prop.

Instructions :

  1. Créez un composant ThemeToggle qui prend une prop onThemeChange (fonction).
  2. ThemeToggle contient un bouton "Activer le mode sombre".
  3. Lorsqu’on clique, il appelle onThemeChange("dark") ou onThemeChange("light").
  4. Dans App, changez dynamiquement l’arrière-plan selon le mode.

🌟 Conclusion

Ces exercices avancés permettent de manipuler des props complexes, d’utiliser des fonctions en props, et de générer dynamiquement des composants. Ils couvrent également des concepts comme le filtrage, les événements, la mise à jour d’état, et la transmission de composants en prop.
