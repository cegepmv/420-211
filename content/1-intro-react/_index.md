------------
React est une **librairie JavaScript** développée par Facebook (Meta) qui permet de créer des interfaces utilisateur **dynamiques, modulaires et performantes**. Elle est particulièrement utilisée pour construire des applications web modernes où l’interface doit réagir rapidement aux changements de données.

L’idée centrale de React repose sur une approche **déclarative** : plutôt que de manipuler directement le DOM (comme en JavaScript traditionnel), on décrit **ce que l’interface doit afficher à un moment donné**, et React se charge de synchroniser l’affichage lorsque les données évoluent.

Cette façon de travailler rend le code plus lisible, plus prévisible et plus facile à maintenir, surtout lorsque les applications deviennent complexes.

Grâce à sa popularité et à son vaste écosystème, React est aujourd’hui largement utilisé pour développer :

+ des applications web dynamiques
+ des sites à page unique (SPA)
+ des applications mobiles **natives** avec **React Native**

## Objectifs

Dans ce module, nous allons commencer par créer une page statique avec React.

Tu te dis peut-être :
  > *"Mais je sais déjà faire des pages statiques…"*

C’est vrai — mais développer avec React demande une façon de penser différente. Avant de construire des applications dynamiques et complexes, il est essentiel d’apprendre à réfléchir en React et à poser des bases solides. Ces fondations permettront ensuite d’écrire du code clair, modulaire et évolutif.
